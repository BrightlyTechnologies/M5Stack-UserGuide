# M5GO HUB 

## DESCRIPTION

The M5GO HUB is a module for M5GO to expand GROVE port added three GROVE ports for user. So you can control 3 GROVE devices with only one M5GO Core at a time. 

## FEATURES

- Instantly Add Three 4-Pins-GROVE ports to Any Compatible Device
- Two Lego installation holes
